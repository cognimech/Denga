Denga

Haskell trading framework

Download .zip Download .tar.gz View on GitHub

Denga is an open source algorithmic trading framework written in Haskell. It is designed to provide:

  • broker API/backtesting independent interface
  • powerful tools for backtesting and technical analysis
  • excellent documentation and examples
Denga is maintained by vlasenkov
